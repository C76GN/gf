# 控制台窗口与内置命令

按下 **F1**（可配置）即可呼出半透明控制台。默认保持全屏覆盖；也可以启用窗口模式，让控制台以可拖拽、可缩放面板呈现。

控制台内置命令：

- `help`：列出所有指令。
- `clear`：清空输出。
- `scene.tree`：只读场景树摘要。
- `scene.node`：只读节点摘要。

控制台会自动接收 `GFLogUtility` 的日志信号并着色显示：Error/Fatal 红色、Warn 黄色、Debug 青色。

界面内置日志标签过滤输入框，支持 `Tab` 补全命令、上下方向键切换输入历史。未知命令也可通过 `suggest_similar_commands()` 给出相似候选。
