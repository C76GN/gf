# GF Framework Wiki

GF Framework Wiki 已按当前 `kernel / standard / packages` 源码结构重排。旧 Wiki 已备份到 `ai_analysis/wiki_backup_2026-05-12/`，本版页面按职责归属组织，便于从框架核心、标准库、官方包和维护实践四条线阅读。

## 推荐阅读路径

1. 新项目先读 `00. 总览与快速开始`、`01. Kernel 架构容器`、`02. 生命周期、装配与依赖`。
2. 写业务通信时读 `03. 消息、事件、命令与查询`。
3. 做场景 UI、输入桥接或表现节点时读 `04. 场景桥接、Controller 与数据绑定`。
4. 查标准库工具时按 `05` 到 `08` 的 Standard 分组阅读。
5. 启用、禁用或删除官方包时读 `09. 官方包总览与包规范` 与 `13. 编辑器工具、访问器与项目常量`。
6. 维护框架或检查迁移风险时读 `14. 最佳实践、维护与测试` 和 `更新日志`。
